user/create


# Here is a curl example
curl --location 'https://{{host}}:{{port}}/user/create' \
--header 'Content-Type: application/x-www-form-urlencoded' \
--data-urlencode 'email=' \
--data-urlencode 'password=' \
--data-urlencode 'whitelabel_name='

To create a new whitelabel you need to make a POST call to the following url :
https://{{host}}:{{port}}/user/create 



# Example Result :

{
  query:{
    offset: 0,
    limit: 50,
    house: [
      "Stark",
      "Bolton"
    ],
  }
}

QUERY PARAMETERS

Field Type Description
email String Valid Email Address
password String Min Length 8
whitelabel_name String Alphanumeric (min lenght 5)
