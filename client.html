user/create


# Here is a curl example
curl --location 'https://{{host}}:{{port}}/user/create' \
--data-urlencode 'firstname=' \
--data-urlencode 'lastname=' \
--data-urlencode 'email=' \
--data-urlencode 'password=' \
--data-urlencode 'token='

To create a new user you need to make a POST call to the following url :
https://{{host}}:{{port}}/user/create 


                
# Example Result Success:

{
    "success": true,
    "user": {
        "id": 26,
        "email": "johndoe@mail.com",
        "whitelabel_id": 1
    }
}
                
            
                
# Example Result Error:

{
     success: false,
        message: 'User with this email johndoe@mail.com already exists.'
}

BODY PARAMETERS

Field Type Description
firstname String
lastname String
email String Valid Email Address
password String Min Length 8
token String Whitelabel Token
